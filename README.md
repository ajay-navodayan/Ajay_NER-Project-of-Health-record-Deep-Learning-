# 🧠 BioBERT NER Project (EHR)

This project focuses on **Named Entity Recognition (NER)** in Electronic Health Records using **BioBERT**.

## 📌 Objective
- Extract medical entities like:
  - Disease
  - Symptoms
  - Treatments

## 📊 Benchmark Reference
- n2c2 2018 → F1: 0.92  
- n2c2 Ensemble → F1: 0.9458  
- i2b2 Hybrid → F1: 0.876  
- NCBI + BioBERT → F1: ~0.85–0.88  

## ⚙️ Tech Stack
- Python  
- Hugging Face Transformers  
- BioBERT  
- Google Colab
